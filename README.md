# 🚗 Smart Parking System (Java Project)

## 📌 Project Overview
This project is a console-based Smart Parking System developed using Java.  
It simulates a real-world parking management system where users can park vehicles, remove them, and calculate parking fees.

---

## 🎯 Objectives
- Manage parking slots efficiently  
- Track vehicle entry and exit  
- Calculate parking fees based on time and vehicle type  
- Maintain parking history  

---

## 🛠️ Technologies Used
- Java  
- Object-Oriented Programming (OOP)  

---

## 🧩 Project Structure

The project consists of the following classes:

### 1. Vehicle.java
- Stores vehicle details (number, type, entry time)

### 2. ParkingService.java
- Handles parking operations  
- Allocates and frees slots  
- Manages parking history  

### 3. FeeCalculator.java
- Calculates parking fee based on:
  - Parking duration  
  - Vehicle type (2W / 4W)

### 4. MainApp.java
- Main program with menu-driven interface  
- Takes user input and performs operations  

---

## ✨ Features
- Park a vehicle  
- Remove a vehicle  
- Display parking status  
- View parking history  
- Automatic fee calculation on exit  

---

## ▶️ How to Run the Project

1. Open terminal in project folder
2. Compile all Java files:
```bash
javac *.java
```
3. Run the program:
```bash
java MainApp
```
